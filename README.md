# 🛒 NerdShelf

Projeto desenvolvido utilizando **HTML**, **CSS** e **JavaScript**, com foco em uma experiência moderna para uma loja temática do universo nerd.

O site possui páginas de navegação, sistema de carrinho, catálogo de produtos e uma interface responsiva.

---

# 📌 Funcionalidades

* ✅ Página inicial estilizada
* ✅ Página de loja com produtos
* ✅ Sistema de carrinho lateral (drawer)
* ✅ Toast de confirmação de ações
* ✅ Navegação entre páginas
* ✅ Layout responsivo
* ✅ Estilização moderna utilizando CSS

---

# 🖥️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

# 🚀 Como Executar o Projeto

## 1. Clone o repositório


## 2. Abra a pasta do projeto

```bash

```

## 3. Execute o projeto

Basta abrir o arquivo `index.html` no navegador.
